## Getting Started

### Install Go

Install from the [Go website](https://golang.org/).

### Run the Server

```bash
go run main.go
```

### Run the Tests

```bash
go test -coverprofile=c.out
```

### Open Code Coverage Report

```bash
go tool cover -html=c.out
```
